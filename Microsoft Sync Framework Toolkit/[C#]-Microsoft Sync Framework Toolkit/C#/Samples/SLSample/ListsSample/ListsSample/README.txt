Overview
--------
This sample allows the user manage his lists.  The generated code is in the
ListSampleEntities.cs and ListSampleOfflineContext.cs.  Most interaction with
the offline sync code occurs in the Model\ContextModel.cs file.

Prerequisites
-------------
The sample has the following prerequisites:

Silverlight 3 Toolkit, November 2009: 
http://silverlight.codeplex.com/releases/view/36060

Sample service included in Microsoft Sync Framework V4 October 2010 CTP.
